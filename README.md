# **Fuzzy Mamdani Code**
### This is a fuzzy code to solve the task of watering duration of plants based on Temperature and Soil Mosture Parameters for Input.

## Input Fuzzy
1. Temperature
2. Humidty

## Output Fuzzy
1. Watering Duration
